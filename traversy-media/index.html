Resources for HTML learning and professional development

Disclaimer

The following list is just a collection of resources I discovered on my way of improving my skills to become a frontend developer. You don't have to use all of them. This is just a combination that works for me personally. The order of resources is alphabetical and there's no comparison of resources.

Resources

  • freeCodeCamp
  • Kamran Ahmed's Web Developer Roadmap
  • Scrimba
  • Traversy Media

Resources review

freeCodeCamp

freeCodeCamp has a lot of resources for HTML, CSS and JS. There are also resources to learn some libraries and practical exercises to practice.

Learn more

Kamran Ahmed's Web Developer Roadmap

A great resource that can be used to understand what topics and technologies you should or can learn in order to become a developer. I use this resource to plan my self-education process.

Learn more

Scrimba

Scrimba has free cources to learn basics of HTML, CSS and JS. There are also cources to learn some libraries and frameworks and paid bootcamps.

Learn more

Traversy Media

Traversy Media is a resource that can help both beginners and advanced developers. Here you can learn more about HTML, CSS, JS, and trends in programming.

Learn more

Summary

It doesn't matter what resources you decide to use for professional development as long as these resources are updated regularly and you practise what you have learned. Also it's worth mentioning that no resource can make you a good developer if you don't put enough effort and work hard.
